Using Selectors in CSS

Cascading Style Sheets, or CSS for short, is a great way to add color and style to your webpage. You can do an array of things such as setting font color, size, and style to designing a layout for how you want your webpage to look. Styling can be a tedious thing, but there's a way to make it less time consuming so you can focus more on your content!

To style a wepbage using CSS, we use something called "Selectors". Selectors are how we can tell what is being styled. There are three main ways to use selectors. You can style your webpage based on element tags, class names, or IDs. If using an element tag, you could list the name of the element followed by a set of brackets. Inside the bracket, you would includw what style you want your elements to be. For example, if I used "h3 {font-color: blue; }", I would be specifying that I want all the h3 headers to be the color blue. Class is another great way for grouping styling together. For example, if I created a pet adoption webpage and I wanted all the dog descriptions to have a green background and all the cat descriptions to have a blue background, I could use class names to group them. To do this, you would add "class="dog-descriptions" onto the elements with the dog descriptions. For cats, you would do the same, just name it "cat-descriptions." They both are written in the same element type but you don't want them to all be the same color so you use class to group them. Class is identified in a CSS file by the "." leading the classname. Ex: .dog-descriptions {background-color: green;}

The third main way of setting styles for your webpage is using an ID selector. The ID selector is used by adding id="mission-statement". If we're making a pet adoption website, I want the mission statement to stand out. However, it uses the same element as other parts of the webpage. To differentiate this, we use ID. We only want one thing to have different styling from the rest. To identify an ID selector, it would include a hash symbol before the element name. Ex: #mission-statement {font-size: 20px; color: red;} Using these different selectors will help you bring your webpage to life! For more on selectors, Mozilla has some great docs to reference!

Why Alts Are Important for Media Content

Using Alts in your webpage might not come naturally, but it's important to make sure they are used. So what are "Alts"? Alts are an attribute added to html elements where Alternate text is used in case the element doesn't render on the page. This is most commonly used in media types such as images and videos.

Why are Alts important? The biggest importance is for making your webpage accessible! For visually impaired visitors of your webpage, they might be using a screen reader to read your content. Screen readers read the text on a screen but not the images. Including descriptive alertnate text ensures they get the full experience of your website. If you have an image that is necessary for a user to understand your content, you need to make sure all users can access it. For more information, visit W3 Docs.

The "HEAD" of Your Webpage

Including a "head" element for your webpage in very important. While it doesn't include any information that is visibly seen on your page, it supplies a lot of information to the internet about your webpage. The "head" tag is placed at the top your your html file before the body. It includes the "title" element. This title element is what is shown on the browser's title bar or the tab your user has open on their web browser. This title will also be what is displayed in search results when your webpage comes up in results which is why it is important to have a good title or the user might think your web page isn't relevant to what they've searched for.

The "style" tag can be used to define the stylings for the single web page. This isn't necessary should you decide to use a css file to define your style. If using an external stylesheet instead of a style tag, a "link" tag is used. The link tag will contain a link the source of your stylesheet.

The "meta" tag will contain a lot more information for the internet to know what your page is. There are different attributes associated with the meta tag to provide different information to the internet. name="keywords" content="blog, html, css, html heads" help the internet know what keywords are associated with your website to better assist search engines with knowing your webpage. name="description" content="This blog is about the different elements nested inside the head of a webpage and why they are important" is the description that will display on the search results when your webpage comes up. There are other attributes associated with the meta tag that are useful. There are also other elements that can be contained in the "head" of a webpage. You can check thatm out at w3 Schools.

Choosing A Color Palette

Choosing the right colors for your website if important! There are a lot of options out there. Some important things to remember when choosing a color palette are:

  • Having high contrast
  • Don't use too many colors
  • Avoid using colors together that are hard to distinguish for the color blind

For high contrast, you don't want to use a light color on a light color or a dark color on a dark color. Using a yellow text with a white background can be hard to read. Choosing to use black text on a white background is a lot easier on the eyes. Avoid using too many colors to make your webpage look too busy. Try to stick to a theme of 3 - 5 colors. There are a lot of resources out there to help choose a good color palette. Make sure you're aware of which colors are considered color blind colors. Red & green are two colors that are common color blind colors so using them together would make it hard for someone with color blindness to read. Don't forget to check the Web Accessibility guidelines to make sure your website is accessible by all! You can check out some color schemes here!
